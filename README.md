# Langchain-GenAi-

 
### [create virtual environment where virtual environment name is .venv]
python -m venv .venv

### [activate virtual environment]
.venv\Scripts\activate


###  install required packages from requirements.txt (for existing projects)
'''This command reads the requirements.txt file and installs every package listed.'''
pip install -r requirements.txt


###  create requirements.txt (for new projects) (after installing packages)
'''This command looks at everything installed in your current Python environment and writes it to requirements.txt.
'''
pip freeze > requirements.txt
